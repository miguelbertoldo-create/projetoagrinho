> Equilíbrio entre Produção e Meio Ambiente.

Este é um projeto web completo e expandido, desenvolvido para apresentar de forma clara e interativa os pilares da agricultura sustentável. O site destaca o uso de tecnologia de ponta no campo, métodos modernos de manejo ecológico, indicadores de preservação ambiental e disponibiliza um canal direto de contacto com especialistas.

---

## 🚀 Funcionalidades

* **Design Responsivo Avançado:** Totalmente adaptado para visualização fluida em smartphones, tablets e computadores.
* **Menu de Navegação Inteligente:** Rolagem suave (*smooth scroll*) entre as seções e cabeçalho fixo (*sticky*) para melhor usabilidade.
* **Contadores Animados Dinâmicos:** Indicadores estatísticos estimados (árvores preservadas, água economizada e CO₂ reduzido) com animação progressiva via JavaScript (60 FPS).
* **Manejo Ecológico Detalhado:** Seções informativas sobre Rotação de Culturas, Plantio Direto e Integração Lavoura-Pecuária-Floresta (ILPF).
* **Formulário de Contacto Interativo:** Validação de dados em tempo real e feedback visual instantâneo para o utilizador sem recarregar a página.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Estilização moderna através de variáveis nativas (`:root`), Grid Layout, Flexbox e transições suaves.
* **JavaScript (ES6):** Lógica assíncrona para manipulação do DOM, animações numéricas baseadas em quadros e interceptação de formulários.

## 📦 Como Executar o Projeto

1. Crie uma pasta no seu computador e adicione os seguintes ficheiros:
   * `index.html`
   * `style.css`
   * `main.js`
   * `LICENSE`
   * `README.md`
2. Copie os respetivos códigos fornecidos para cada ficheiro.
3. Abra o ficheiro `index.html` diretamente em qualquer navegador web moderno (Chrome, Firefox, Edge, Safari).
