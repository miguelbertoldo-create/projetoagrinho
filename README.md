# 🌱 Agro Forte, Futuro Sustentável

> Equilíbrio entre Produção e Meio Ambiente.

Este é um projeto web desenvolvido para apresentar os pilares da agricultura sustentável, destacando o uso da tecnologia no campo, indicadores de preservação ambiental e práticas que unem produtividade e conservação do planeta.

---

## 🚀 Funcionalidades

* **Design Responsivo:** Adaptado para telas de celulares, tablets e computadores.
* **Menu de Navegação Suave:** Rolagem fluida entre as seções da página.
* **Contadores Animados:** Indicadores estatísticos (árvores preservadas, água economizada e CO₂ reduzido) que aumentam progressivamente usando JavaScript.
* **Interface Limpa e Moderna:** Uso de componentes visuais (cards) para facilitar a leitura.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica da página.
* **CSS3:** Estilização, layout em Flexbox e responsividade.
* **JavaScript (ES6):** Lógica para animação dos contadores numéricos.
